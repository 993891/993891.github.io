高低位交换：

题目描述

给出一个小于232的正整数。这个数可以用一个32位的二进制数表示（不足32位用0补足）。我们称这个二进制数的前16位为“高位”，后16位为“低位”。将它的高低位交换，我们可以得到一个新的数。试问这个新的数是多少（用十进制表示）。

例如，数1314520用二进制表示为00000000000101000000111011011000（添加了11个前导0补足为32位），其中前16位为高位，即0000000000010100；后16位为低位，即0000111011011000。将它的高低位进行交换，我们得到了一个新的二进制数00001110110110000000000000010100。它即是十进制的249036820。

输入格式

一个小于232的正整数

将新的数输出

    unsigned int n,nl;
    cin>>n;
    nl=n;
    n=n<<16;//左移动 低16位去高16位了
    nl=nl>>16;//右移动 高16位变低16位
    cout<<n+nl;

互换奇偶位:

题目描述

给定一个int类型正整数，请将其二进制上的奇偶位进行互换，如0110−>1001

输入格式

一个int类型正整数n

n<= 108

思路：

    10111110

&  01010101 −> 0x5555555

      ----- -----

        5 5

    10101010 −> 0xaaaaaaa

    int n,a,b;
    cin>>n;
    a=n&0x55555555;//取奇数位置
    b=n&0xaaaaaaaa; //取偶数位置
    int ans=(a<<1)+(b>>1);
    cout<<ans;
    return 0;
